Installed plugins

Plugin manifest format
  1. id, name, version, description
  2. permissions: project:read, project:write, ui:notify, tests:run, preview:run
  3. code: JavaScript receiving a scoped api object
  4. Register commands with api.registerCommand(...)
  5. Register hooks with api.on('beforePreview' | 'afterPreview' | 'beforeTests' | 'afterTests' | 'projectChanged', fn)
